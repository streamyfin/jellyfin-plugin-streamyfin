These settings are the ones found on the application itself. As the admin you are allowed to set default values to these settings and also choose to lock them from your users.

Video settings

The amount of time in seconds you want to be able to rewind during playback
Audio settings
Allows you to set the audio language from the previous played item
Subtitle settings
Setting to determine when subtitles will automatically play during video playback
Allows you to set the subtitle language based on the previous played item
Adjust the subtitle size during video playback
